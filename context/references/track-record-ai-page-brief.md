## Page 2 — Applied AI ("Applied AI – Real Workflows in Investment Process")

### Core principle
NOT "I built AI tools"
SELLING "I systematically improve research process using AI → better decisions, faster"

### Standard: must pass the "30-second live demo" test
Can I open this on my laptop and show it live in 30 seconds? If yes → credible.

### Structure (3 concrete artefacts)
Each block: Input → Prompt → Output (screenshot) → Decision impact

**Block 1: Idea Screening Tool (Shorts)**
- Input: 15–20 screened names (Excel export)
- Prompt: rank based on crowding, earnings risk, momentum deterioration
- Output screenshot: ranked table + short rationale
- Impact: identified [X] as top short candidate

**Block 2: Earnings Call Decomposition**
- Input: transcript
- Output: what changed vs last quarter, key drivers, miss risks
- Impact: faster identification of inflection / miss risk

**Block 3: Variant View Builder**
- Input: company + consensus assumptions
- Output: structured Bull/Bear/Base + key debate points
- Impact: sharper thesis articulation for PM discussion

### What NOT to do
- No fake dashboards or UI mockups
- No apps (people will test them, ask technical questions)
- No generic ChatGPT outputs without framing
